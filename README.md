# Diwali_sales_analysis_python_project
1:Project Objective
Analyzed Diwali sales data to understand customer behavior, product performance, and sales trends to support data-driven business decisions.

2:Dataset Description
Used a structured CSV dataset containing customer demographics, product categories, order details, and sales values during the Diwali season.

3:Tools & Technologies
Python, Pandas, NumPy for data cleaning and analysis; Matplotlib and Seaborn for data visualization; Jupyter Notebook for implementation.

4:Key Analysis Performed
Conducted Exploratory Data Analysis (EDA) to identify top-selling products, high-revenue states, gender-wise spending patterns, and age-group trends.

5:Results & Insights
Discovered key customer segments and sales drivers that can help businesses optimize marketing strategies and improve Diwali season revenue.
